# Sobre o trabalho

A linguagem de programação utilizada para criar as classes foi e as funcionalidades foi a linguagem`JAVA`.

# Requisitos

Para poder executar o código é preciso ter instalado:

- JAVA 21(JDK)
- GRANDLE (Opcional)

# Comandos

Após ter instalado java basta rodar os seguintes comandos:
```
./gradlew build
java -jar app/build/libs/app.jar
```
